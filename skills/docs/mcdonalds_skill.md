# McDonald's Skill - 麦当劳服务

## 描述
提供麦当劳餐厅查询、菜单浏览、营养计算和促销活动查询服务。

## 适用场景
当用户需要以下信息时调用此技能：
- 查找附近的麦当劳餐厅
- 浏览麦当劳菜单和价格
- 了解特定餐品的营养信息
- 查看当前促销活动
- 计划麦当劳就餐

## 工具函数

### find_nearby_restaurants

查找附近的麦当劳餐厅。

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| latitude | float | 是 | 纬度，例如 31.2304（上海） |
| longitude | float | 是 | 经度，例如 121.4737（上海） |
| radius | int | 否 | 搜索半径（米），默认 5000 |
| limit | int | 否 | 返回结果数量，默认 10 |

**使用示例：**
```python
# 查找上海南京东路附近的麦当劳
find_nearby_restaurants(
    latitude=31.2304,
    longitude=121.4737,
    radius=3000,
    limit=5
)
```

### get_menu

获取麦当劳菜单。

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| category | string | 否 | 菜单分类，可选值：burger(汉堡), chicken(鸡肉), fries(小食), drink(饮品), dessert(甜品), breakfast(早餐), happy_meal(开心乐园餐)。不传则返回所有分类 |

**使用示例：**
```python
# 获取汉堡菜单
get_menu(category="burger")

# 获取早餐菜单
get_menu(category="breakfast")

# 获取完整菜单
get_menu()
```

### get_meal_categories

获取所有套餐分类列表。

**使用示例：**
```python
get_meal_categories()
```

### calculate_nutrition

计算餐品的营养信息。

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| items | list | 是 | 餐品名称列表，例如 ["巨无霸", "薯条(大)", "可乐(中)"] |

**使用示例：**
```python
# 计算套餐营养
calculate_nutrition(items=["巨无霸", "薯条(中)", "可乐(中)"])

# 计算多件商品
calculate_nutrition(items=["麦辣鸡腿汉堡", "麦辣鸡翅(2块)", "麦旋风(Oreo)"])
```

### check_promotions

查询当前促销活动。

**使用示例：**
```python
check_promotions()
```

- 餐厅数据为模拟数据，实际项目中应接入真实 API

- 早餐菜单仅在早餐时段供应（通常 05:00-10:30）
